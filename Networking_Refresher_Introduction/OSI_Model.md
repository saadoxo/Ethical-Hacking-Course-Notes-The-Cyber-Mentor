# OSI Model Cheat Sheet

A common mnemonic to remember the OSI Model layers:

**"Please Do Not Throw Sausage Pizza Away."**

Each word represents a layer from 1 to 7:

1. **P** - **Physical Layer**  
2. **D** - **Data Link Layer**  
3. **N** - **Network Layer**  
4. **T** - **Transport Layer**  
5. **S** - **Session Layer**  
6. **P** - **Presentation Layer**  
7. **A** - **Application Layer**
